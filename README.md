# Introduction

Feane - A comprehensive platform, that provides customer services and restaurant management. The website focus on simplify with mordern UI, easy-to-use, and safely security

**Customers**
    - Browse menu, food information about: price, promotions, description,...
    - Reservate table online quickly and instantly
    - Order food and track delivery status

**Restaurant**
    - Update menu, price and promotions
    - Manage reservations and avoid double-booking
    - Manage staff, schedule and track work progress
    - Access to customer and financial analysis

## Dependencies

Feane is a fullstack project, that developed and built on NodeJs.

**- Markup & Style languages: HTML/CSS**
**- Programming Language: TypeScript**
**- Runtime/Platform: Node.js**
**- Version Control & Collaboration: Git, GitHub**
**- Tools: ESLint**

1. Frontend
    - Framework/Libary: Next.js
    - Styling Framework: Tailwind
    - UI Component: Shadcn UI
    - State Management: Redux Toolkit
    - Networking: Axios
    - Form: React Hook Form
    - Validation: Zod

2. Backend
    - Framework/Libary: NestJS
    - Database: PostgreSQL
    - ORM: Prisma
    - Authentication & Authorization: JWT
